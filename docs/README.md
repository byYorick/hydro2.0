Документация и спецификации.

**ВАЖНО:** Эталонная документация находится в `doc_ai/`.  
Эта папка содержит:
- Дополнительные документы, специфичные для разработки (testing/, mobile/)
- Копии некоторых документов для удобства доступа

**Основная документация:**
- `doc_ai/` — эталонная документация (не изменяется)
- `doc_ai/INDEX.md` — главный индекс документации

**Структура:**
- `01_OVERVIEW/` — обзор системы
- `02_HARDWARE_FIRMWARE/` — документация по железу и прошивкам
- `03_TRANSPORT_MQTT/` — MQTT протокол

- `05_MOBILE/` — документация мобильного приложения
- `06_INFRA/` — инфраструктура и деплой
