## Как использовать референсы

1. **При проектировании новых страниц:**
   - Изучите референсы перед началом работы
   - Выберите подходящие паттерны для конкретной страницы
   - Адаптируйте под специфику проекта (см. `../../doc_ai/07_FRONTEND/FRONTEND_UI_UX_SPEC.md`)

2. **При разработке компонентов:**
   - Используйте референсы как источник вдохновения для:
     - Расположения элементов
     - Цветовых схем
     - Типографики
     - Иконографии

3. **При создании дашбордов:**
   - Комбинируйте подходы из разных референсов
   - Учитывайте специфику ролевых дашбордов (Agronomist, Engineer, Operator)

## Принципы адаптации

При использовании референсов учитывайте:

1. **Стиль проекта:** минималистичный, вдохновлённый Vercel/Dashboard, shadcn/ui, Linear.app, Home Assistant
2. **Цветовая схема:** тёмная тема по умолчанию, акцентные цвета только для статусов
3. **Технологии:** Laravel + Inertia + Vue 3 + Tailwind CSS
4. **Адаптивность:** обязательная поддержка мобильных устройств
