# Исправление кеша браузера

## Проблема

Браузер использует старый production build (`app-D6XbxRD0.js`) вместо нового (`app-BP6mrxP2.js`).

## Решение

1. **Очистить кеш браузера**:
   - Firefox: Ctrl+Shift+Delete → выберите "Кеш" → очистить
   - Или жесткая перезагрузка: Ctrl+Shift+R (или Ctrl+F5)

2. **Проверить, что новый build используется**:
   - Откройте DevTools (F12)
   - Перейдите в Network → обновите страницу
   - Найдите `app-*.js` файл
   - Должен быть `app-BP6mrxP2.js`, а не `app-D6XbxRD0.js`

3. **Если проблема сохраняется**:
   - Очистите кеш Laravel: `php artisan view:clear`
   - Проверьте, что файл существует: `ls -lh public/build/js/app-*.js`
