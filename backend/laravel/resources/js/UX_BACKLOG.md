# UX Backlog

Список текущих UX-болей и целей улучшения.

- Analytics: страница заглушка и стилистически выбивается — `backend/laravel/resources/js/Pages/Analytics/Index.vue`.
- Alerts: фильтры бедные, нет деталей и группировки, не использует store/API полноценно — `backend/laravel/resources/js/Pages/Alerts/Index.vue`.
- Zone detail: перегружен, много блоков на одном экране, сложно расширять — `backend/laravel/resources/js/Pages/Zones/Show.vue`.
