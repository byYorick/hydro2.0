# Сводка ошибок сборки

## Проблема

Docker build падает при сборке frontend с ошибкой парсинга Vue файлов:
- `Pages/Devices/Add.vue` - ошибка на строке 294-297
- `Pages/Devices/Show.vue` - похожие ошибки

## Причина

Vue компилятор не может правильно обработать многострочные выражения с проверками свойств объектов в catch блоках.

## Решение

Использовать простую однострочную проверку или вынести логику в отдельную функцию.

## Статус

Исправления применены, но требуют пересборки Docker без кэша.
