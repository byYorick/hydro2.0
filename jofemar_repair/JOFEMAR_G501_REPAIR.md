# Замена микроконтроллера PIC24HJ256GP206 на плате Jofemar G-501

**Плата:** Jofemar G-501 (CPU board кофейного автомата Coffeemar G-546D)  
**Микроконтроллер:** PIC24HJ256GP206 (Microchip, 16-bit, TQFP-64)  
**Кварц:** 8 МГц  
**Прошивка:** G501_F14 (версия F14)  
**Дата:** 2026-04-17

---

## 1. Необходимые компоненты и инструменты

### Микроконтроллер

| Параметр | Значение |
|----------|----------|
| Чип | **PIC24HJ256GP206** или **PIC24HJ256GP206A** |
| Корпус | TQFP-64 (10x10 мм, шаг 0.5 мм) |
| Где купить | Mouser, DigiKey, LCSC, ChipDip, AliExpress |

### Программатор

| Вариант | Цена | Примечание |
|---------|------|------------|
| **Клон PICkit 3** (AliExpress) | ~1000 руб | Для разовой задачи — достаточно. PIC24HJ работает на 3.3В, проблемы клонов с 5В не актуальны |
| **MPLAB SNAP** (PG164100) | ~$15-35 | Оригинал Microchip, без корпуса |
| **PICkit 4** (PG164140) | ~45000 руб | Лучшая совместимость, дорого для разового ремонта |

### Паяльное оборудование

- Паяльная станция с феном (hot air rework)
- Флюс пастообразный (RMA-223 или аналог)
- Оплётка для припоя
- Изопропиловый спирт (IPA) для промывки
- Лупа или микроскоп
- Антистатический браслет

### ПО

- **MPLAB X IDE** + **MPLAB IPE** (скачать с сайта Microchip бесплатно)
- Для клона PICkit 3 использовать MPLAB X **v6.15–v6.20** (в новых версиях поддержку PICkit 3 убрали)

## 2. Демонтаж старого чипа

1. Нанести флюс на все 64 вывода
2. Прогревать феном равномерно по кругу: **350–380°C**, средний поток воздуха
3. Когда припой оплавится — аккуратно снять чип пинцетом
4. Не грей дольше 30–40 секунд — риск повреждения платы и соседних компонентов
5. Очистить площадки оплёткой с флюсом
6. Промыть остатки флюса изопропиловым спиртом

---

## 3. Установка нового чипа

1. Нанести флюс на площадки
2. Выровнять чип по ключу (**точка/скос на корпусе = pin 1**)
3. Припаять:
   - **Вариант с феном:** прогревать сверху ~300–320°C, припой сам затечёт
   - **Вариант с паяльником:** сначала 2 угловых вывода для фиксации, затем остальные
4. Проверить под лупой/микроскопом — **нет замыканий** между ножками
5. Прозвонить мультиметром: VDD не замкнут на GND

## 4. Подключение программатора (ICSP)

### Распиновка PIC24HJ256GP206 (TQFP-64)

| Сигнал ICSP | Пин чипа | Как найти на разъёме |
|-------------|----------|---------------------|
| **MCLR** | Pin 18 | Обычно через резистор ~10К к VDD |
| **PGC** (PGEC1) | Pin 23 | Прозвонить напрямую к разъёму |
| **PGD** (PGED1) | Pin 24 | Прозвонить напрямую к разъёму |
| **VDD** (3.3V) | Pin 10, 26, 38, 51 | Любой — прозвонить на плюс питания |
| **GND** | Pin 9, 25, 41, 56 | Любой — прозвонить на минус/корпус |

### Подключение PICkit к плате

На плате есть 12-контактный разъём. Нужно мультиметром в режиме прозвонки определить какие контакты разъёма идут к каким пинам PIC24:

1. **Найти GND** — прозвонить контакты разъёма на минус питания / металлический корпус
2. **Найти VDD** — прозвонить на 3.3В цепь
3. **Найти MCLR** (pin 18) — обычно через резистор, может иметь конденсатор на землю
4. **Найти PGC** (pin 23) — прямое соединение
5. **Найти PGD** (pin 24) — прямое соединение

### Подключение проводов PICkit

```
PICkit Pin 1 (MCLR) ──── MCLR (Pin 18 чипа)
PICkit Pin 2 (VDD)  ──── VDD 3.3V (Pin 10/26/38/51)
PICkit Pin 3 (GND)  ──── GND (Pin 9/25/41/56)
PICkit Pin 4 (PGD)  ──── PGD1 (Pin 24)
PICkit Pin 5 (PGC)  ──── PGC1 (Pin 23)
PICkit Pin 6         ──── не подключать
```

## 5. Прошивка через MPLAB IPE

### Файлы прошивки

| Файл | Описание |
|------|----------|
| `G501_F14.b00` | Оригинал Jofemar, банк 0 (бинарный) |
| `G501_F14.b01` | Оригинал Jofemar, банк 1 (бинарный) |
| `G501_F14_b00.hex` | **Сконвертированный банк 0** — Intel HEX для MPLAB IPE |
| `G501_F14_b01.hex` | **Сконвертированный банк 1** — Intel HEX для MPLAB IPE |

Оба банка — полные самостоятельные прошивки с одинаковой таблицей прерываний
(IVT), но разным основным кодом. **Начать с `G501_F14_b00.hex`.**

### Пошаговая инструкция

1. Запустить **MPLAB IPE**
2. **Device** → выбрать `PIC24HJ256GP206`
3. **Tool** → выбрать подключённый PICkit
4. **Settings → Advanced Mode** (пароль: `microchip`)
5. **Power:**
   - Поставить галку **"Power target circuit from PICkit"**
   - Напряжение: **3.3V**
6. **File → Import** → выбрать `G501_F14_b00.hex`
7. **Settings → Configuration Bits** → ввести значения из таблицы ниже
8. Нажать **Program**
9. Дождаться сообщения "Programming complete"
10. Отключить PICkit, подать питание на плату, проверить работу

### Если b00 не работает

- Повторить шаги 6–10 с файлом `G501_F14_b01.hex`

## 6. Configuration Bits

**КРИТИЧНО:** Без правильных config bits чип не запустится!

В бинарных файлах Jofemar конфигурационные регистры **не содержатся** —
их нужно задать вручную в MPLAB IPE.

### Рекомендуемые значения (8 МГц кварц + PLL)

| Регистр | Hex | Настройки |
|---------|-----|-----------|
| **FOSCSEL** | `0x0003` | FNOSC=PRIPLL (Primary Oscillator + PLL), IESO=OFF |
| **FOSC** | `0x00E2` | FCKSM=CSDCMD (clock switching disabled), OSCIOFNC=OFF, POSCMD=HS |
| **FWDT** | `0x005F` | FWDTEN=OFF (watchdog отключён — для первого теста!) |
| **FPOR** | `0x0087` | FPWRT=PWR128 (стандартный power-on reset delay) |
| **FGS** | `0x0007` | GSS=OFF, GCP=OFF (без защиты кода) |
| **FBS** | `0x000F` | BSS=OFF, BWRP=OFF (без защиты boot-сегмента) |
| **FICD** | `0x0003` | ICS=PGD1 (ICSP через PGD1/PGC1), JTAGEN=OFF |

### Расшифровка осциллятора

- **POSCMD=HS** — режим High-Speed для кварца 8 МГц
- **FNOSC=PRIPLL** — Primary Oscillator с PLL-множителем
- PLL-множитель и делители (CLKDIV, PLLFBD) задаются **программно в коде прошивки**,
  не в config bits. Прошивка Jofemar сама настраивает PLL при старте.
- 8 МГц × PLL → до 80 МГц системной частоты (40 MIPS)

### Если плата не стартует

Возможные причины и что пробовать:

1. **Неверный режим осциллятора** — попробовать POSCMD=EC (External Clock) вместо HS:
   - FOSC = `0x00E0` (EC mode)
2. **Нужен FRC+PLL вместо Primary+PLL** — попробовать:
   - FOSCSEL = `0x0001` (FNOSC=FRCPLL)
3. **Watchdog сбрасывает** — убедиться что FWDTEN=OFF
4. **Проблема с пайкой** — проверить все 64 контакта под микроскопом
5. **Неверный банк прошивки** — попробовать второй .hex файл

## 7. Альтернативные способы восстановления

### Способ Б: Считать прошивку с рабочей платы-донора

Если есть доступ к второму рабочему автомату Coffeemar G-546D:
1. Подключить PICkit к ICSP на рабочей плате
2. MPLAB IPE → Read → сохранить полный дамп (включая config bits!)
3. Этот дамп залить в новый чип — самый надёжный способ (config bits будут правильные)

**Условие:** code protection должен быть отключён (GCP=OFF, GSS=OFF). Если включён — считать не получится.

### Способ В: Jofemar EasyFlash

Штатный инструмент Jofemar для прошивки через RS-232:
1. Нужен оригинальный **Jofemar EasyFlash v2** или ПО **Jofemar Flash Programmer**
2. Сначала нужен bootloader в чипе (замкнутый круг без ICSP)
3. Подходит только если bootloader уже записан

### Способ Г: Купить б/у плату

Площадки запчастей Jofemar (Испания):
- [Fersumac](https://fersumac.net/73-electronica-jofemar) — электроника Jofemar
- [MaxVending](https://www.maxvending.es/) — запчасти вендинг
- [maquinasdetabaco.com](https://www.maquinasdetabaco.com/) — запчасти Jofemar
