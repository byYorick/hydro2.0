# Climate Node

Климатическая нода для измерения температуры, влажности, CO₂ и управления климатическим оборудованием.

## Описание

Нода измеряет параметры климата (температура, влажность, CO₂, освещённость) и управляет вентиляторами, нагревателями, освещением.

## Каналы

### Сенсоры
- `temperature` - температура воздуха (SHT3x)
- `humidity` - влажность воздуха (SHT3x)
- `co2` - концентрация CO₂ (CCS811/SGP30, опционально)
- `lux` - освещённость (опционально)

### Актуаторы
- `fan_A`, `fan_B` - вентиляторы
- `heater_1` - нагреватель
- `white_light`, `uv_light` - освещение

## Команды

- `set_relay` - включение/выключение реле (вентилятор, нагреватель)
- `set_pwm` - установка PWM значения (вентилятор, освещение)
